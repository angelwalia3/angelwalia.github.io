i-Woz

Smart Table Lamp

Introduction
i-Woz is a lamp that not only functions as a traditional lamp, but also allows you to completely move each of its hinges to set its angle with an app. It comes with two special modes, Deadline and Ambience. The Deadline mode allows us to add our deadlines and checklists with a time period. As the clock inches closer and closer to instances on the checklist or towards the deadline, The light of the lamp changes from white to red. Till the checklist is checked off and till the deadline is completed, this light functions as a warning system as well. Ambience mode allows the user to customise the lights for a more mood related setting.

3D Design and Prototype Lamp

The App
The app was designed through MIT App Inventor. The lamp is controlled entirely through this app. The lamp can be oriented using the buttons that control the direction of each motor, one to move it up and one to move it down.
Ambience mode has many mood and fun lighting options built-in.
Deadline mode allows us to select the time for the deadline and feed this input to the lamp.
